$Dump Shaders	Dump Shaders
$Dump Shaders Description	起動時にシェーダーをダンプします。これはシェーダーをリバースエンジニアリングする場合にのみ使用してください。通常のユーザーには必要ありません。
$Log Level	ログレベル
$Log Level Description	ログレベル。トレースが最も詳細です。デフォルトは情報です。
$Shader Defines	シェーダー定義
$Shader Defines Description	シェーダーコンパイラ用の定義。セミコロン「;」で区切ります。スペースでクリアします。変更後にシェーダーを再構築してください。コンピュートシェーダーは再コンパイルのために再起動が必要です。
$Compiler Threads	コンパイラスレッド
$Compiler Threads Description	シェーダーをコンパイルするために使用するスレッド数。スレッド数が多いほどコンパイルは速く終了しますが、システムの応答性が低下する可能性があります。
$Background Compiler Threads	バックグラウンドコンパイラスレッド
$Background Compiler Threads Description	ゲームをプレイ中にシェーダーをコンパイルするために使用するスレッド数。これは起動時のコンパイルがスキップされた場合に有効になります。スレッド数が多いほどコンパイルは速く終了しますが、システムの応答性が低下する可能性があります。
$Test Interval	テスト間隔
$Test Interval Description	デフォルトのUSERとTEST設定間を切り替える秒数を設定します。0は無効化します。非ゼロの場合はテストモードを有効にします。これにより現在の設定がTEST設定として保存されます。設定が変更されていない場合は影響しません。
$Enable File Watcher	ファイルウォッチャーを有効化
$Enable File Watcher Description	ファイル変更時にシェーダーを自動的に再コンパイルします。開発用です。
$Dump Ini Settings	Ini設定をダンプ
$Stop Blocking Shaders	{}シェーダーのブロッキングを停止
$Stop Blocking Shaders Description	Community Shadersのシェーダーのブロッキングを停止します。ブロッキングはゲーム内でシェーダーエラーをデバッグする際に、どのシェーダーに問題があるかを特定するのに役立ちます。開発者モードでPAGEUPおよびPAGEDOWNを押すとブロッキングが有効になります。特定のシェーダーがログファイルに出力されます。
$Statistics	統計
$Replace Original Shaders	オリジナルシェーダーを置き換え
$Vertex Shader Description	頂点シェーダーを置き換えます。無効の場合、上記のタイプのカスタム頂点シェーダーが無効になります。開発者がCSシェーダーがバニラの動作と一致するかをテストするために使用します。
$Pixel Shader Description	ピクセルシェーダーを置き換えます。無効の場合、上記のタイプのカスタムピクセルシェーダーが無効になります。開発者がCSシェーダーがバニラの動作と一致するかをテストするために使用します。
$Compute Shader Description	コンピュートシェーダーを置き換えます。無効の場合、上記のタイプのカスタムコンピュートシェーダーが無効になります。開発者がCSシェーダーがバニラの動作と一致するかをテストするために使用します。
